## What you can do

- Write posts in `_posts/` as `YYYY-MM-DD-title.md`
- Add front matter with `title`, `date`, and optional `author`
- Use any Markdown formatting: **bold**, *italic*, `inline code`, links, images, and more

## Code example

```ruby
def greet(name)
  puts "Hello, #{name}!"
end

greet("World")
```
